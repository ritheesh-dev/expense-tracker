# Expense Tracker with Visualization

A simple Python project to track expenses and visualize them using charts.

## 🚀 Features
- Add, update, and delete expenses
- Store data in MySQL database
- Visualize expenses with charts (Matplotlib/Seaborn)

## 🛠️ Tech Stack
- Python
- MySQL
- Matplotlib

## 📂 Project Structure
- `expense_tracker_mysql.py` → main program
- `.gitignore` → ignored files
- `README.md` → project documentation
